belajar flexbox

flexbox adalah model tata letak 1 dimensi untuk mengatur model item secara horizontal atau vertikal didalam sebuah wadah(div)

komponen flexbox adalah kontainer flexsibel dimana properti display diberi nilai flex.

properti kontainer css flex

  1. display: harus diisi dengan flex atau inline-flex.
  2. flex-direction: mengatur arah tampilan item flex.

    3. flex-direction juga mempunyai banyak nilai,antara lain:
    • row(default)- tugasny adalah menampilkan item flex secara horizontal dari kiri ke kanan.
      • contoh:
      1
      2
      3
      4
    • column- tugasnya menampilkan item flex secara vertikal dari atas ke bawah.
      • contoh:
      1
      2
      3
      4
    • row-reverse - tugasnya menampilan item flex secara horizontal tetapi arahnya dari kanan ke kiri.
      • contoh:
      1
      2
      3
      4
    • column-reverse - tugasnya menampilkan item flex secara vertikal tetapi arahnya dari bawah ke atas.
      • contoh:
      1
      2
      3
      4

  3. flex-wrap: menentukan apakah item flex harus membungkus atau tidak.
    4. flex-wrap juga mempunyai banyak nilai antara lain:
    • nowrap(default) - tugasnya menentukan bahwa item flex tidak akan membungkus.
      • contoh:
      1
      2
      3
      4
      5
      6
      7
    • wrap - tugasnya menentukan bahwa item flex akan membungkus ketika diperlukan.
      • contoh:
      1
      2
      3
      4
      5
      6
      7
    • wrap-reverse - tugasnya menentukan bahwa item flex akan membungkus ketika diperlukan,tetapi dalam keadaan terbalik.
      • contoh:
      1
      2
      3
      4
      5
      6
      7

  4. flex-flow: singkatan untuk flex-direction dan flex-wrap.

  5. justify-content: menyelaraskan item flex ketika item tersebut tidak menggunakan seluruh ruang yang tersedia pada sumbu utama(horizontal/garis lurus dari kiri ke kanan.)
    6. justify-content memiliki beberapa nilai diantaranya:
    • center - tugasnya menyelaraskan item flex ditengah kontainer.
      • contoh:
      1
      2

    • flex-start - tugasnya menyelaraskan item flex diawal kontainer.
      • contoh:
      1
      2

      7
    • space-around - tugasnya menampilkan item flex dengan spasi disekitarnya.
      • contoh:
      1

    • flex-end - tugasnya menyelaraskan item flex diakhir kontainer.
      • contoh:
      1

    • space-between - tugasnya menampilkan item flex dengan spasi diantaranya.

      1

    • space-evenly - tugasnya menampilkan item flex dengan jarak yang sama disekitarnya.

      7
  6. align-items: menyelaraskan item flex ketika item tersebut tidak menggunakan seluruh ruang yang tersedia pada sumbu silang(vertikal/atau dari atas kebawah).
    • center - tugasnya menyelaraskan item flex ditengah kontainer.
    • flex-star - tugasnya menyelaraskan item flex di bagian atas kontainer.
    • flex-end - tugasnya menyelaraskan item flex dibagian bawah kontainer.
